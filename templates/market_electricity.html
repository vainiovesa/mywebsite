Tässä projektissa käytän ENTSO-E Transparency Platform API-rajapintaa Suomen pörssisähkön hinnan selvittämiseen. Hinnan avulla kontrolloidaan liikennevaloja, joista näkee kätevästi, kannattaako sähköä kuluttaa juuri nyt. Otin liikennevalot tänne näytille ja niiden lähdekoodi on nähtävillä GitHub -tililläni.

Liikennevalot toimivat seuraavasti:

  • Kun hinta on alhainen, palaa vihreä valo.
  • Jos hinta ylittää {{ info.yb }} senttiä, syttyy keltainen valo.
  • Jos hinta ylittää {{ info.rb }} senttiä, palaa punainen valo.

Hinta haettiin {{ info.ct }}, jolloin hinta oli {{ info.price }} senttiä / kWh.

{% elif session.language == 'en' %}

In this project, I use the ENTSO-E Transparency Platform API to retrieve the market electricity price in Finland. The price is used to control a set of traffic lights that conveniently indicate whether it's a good time to consume electricity. I've brought the traffic lights here for display and you can view their source code on my GitHub.

The traffic lights operate as follows:

  • When the price is low, the green light is on.
  • If the price exceeds {{ info.yb }} cents, the yellow light turns on.
  • If the price exceeds {{ info.rb }} cents, the red light appears.

The price was fetched at {{ info.ct }}, at which time the price was {{ info.price }} cents / kWh.
